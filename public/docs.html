Job - Deploy

Deploys a job to the DCP scheduler for computation. Only required fields are and . Currently supported languages are JavaScript and TypeScript, but more are coming soon!

Authorizations:
bearerAuth
Request Body schema: application/json
required
object
object
Array of objects or object
args
Array of strings

extra parameters to pass to the workfunction's arguments. Added to the end of the arguments to call the work function

Array of objects
slicePaymentOffer
string

which is a decimal number or MARKET_VALUE

requirements
object

An object that describes the requirements a worker must have to be eligible for the job.

packages
Array of strings

list of packages to use for this job

object

http://bestia.office.distributive.network/api/job
http://rhea.office.distributive.network/api/job

bearerAuth

Responses
